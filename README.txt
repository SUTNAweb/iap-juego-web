🚀 Cómo subir este juego a Vercel:

1. Ve a https://vercel.com
2. Crea una cuenta (puede ser con tu cuenta de GitHub o Google).
3. Haz clic en 'Add New Project' y luego 'Import Project'.
4. Sube este ZIP o sincroniza desde GitHub si prefieres.
5. ¡Listo! Vercel desplegará automáticamente el juego.

Recuerda que al finalizar cada juego, los resultados se enviarán a tu Google Sheets.
